Rodzaje Selektorów w CSS

1. Selektory proste

  • Selektor typu
    • Przykład: p { color: red; }
    • Stosowany do elementów określonego typu, np. akapity, nagłówki.

2. Selektory złożone

  • Selektor uniwersalny
    • Przykład: * { color: red; }
    • Dotyczy wszystkich elementów na stronie.
  • Selektor potomka
    • Przykład: p b { color: red; }
    • Dotyczy elementów potomnych znajdujących się wewnątrz innych elementów.
      • Przykład zastosowania:

        tekst

  • Selektor dziecka
    • Przykład: p > span { color: red; }
    • Dotyczy tylko bezpośrednich dzieci danego elementu.
  • Selektor rodzeństwa
    • Przykład: p + span { color: red; }
    • Dotyczy elementów, które są bezpośrednio sąsiadami innych elementów.

3. Selektory atrybutów

  • Selektor prosty atrybutu
    • Przykład: p[title] { color: red; }
    • Dotyczy elementów posiadających określony atrybut.
  • Selektor atrybutu z kilkoma warunkami
    • Przykład: p[title][lang] { color: red; }
    • Dotyczy elementów, które posiadają wszystkie określone atrybuty.
