Pojęcie Znaczenie pojęcie uzupełnia uczeń
Co znaczy skrót ID? ID to identyfikator — unikalna nazwa przypisana do elementu HTML.
Jaki jest cel stosowania ID? Stosujemy ID, aby wyróżnić i zróżnicować pojedynczy, unikalny element na stronie.
Jakie jest znaczenie znaku # Znak # służy do definiowania ID w arkuszu stylów CSS.
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie ID w HTML następuje przez atrybut: id="nazwa_id". W CSS używamy go z #, np. p#pochyly.
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia.
  1. ID powinno być unikatowe na stronie — nie należy go powielać.
  2. W przypadku wątpliwości co do liczby wystąpień lepiej użyć klasy (class).
  3. Chociaż możliwe jest użycie tego samego ID więcej niż raz, jest to błędem projektowym.
